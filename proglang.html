Operating Systems Terminology
Terms Definition
Machine language Machine lauaguage is language, consisting of binary or hexadecimal instructions, undertood by computers. It's the only langage that a computer can work with.
Debugging Debugging is the process of finding and resolving bugs within computer programs, software, or systems.
Assembler An assembler is a program that converts assemly language into machine code.
Assembly Language Assembly language is a low-level programming language that can communicate directly with the computer's hardware.
Translator A translator is a programming language processor that converts a computer program from one language to another.
Compiler A compiler is a special program that translates a programming language's source code into machine code, bytecode or another programming language.
Interpreter An interpreter is a computer program that is used to directly execute program instructions written using one of the many high-level programming languages.
Cross-Platform Software Cross-platform software is software that can run on multiple operating systems, including Windows, Mac OS, iOS, and Android.
Natural Language Natural language processing (NLP) is the ability of a computer program to understand human language as it is spoken and written
Formal Language A finite or infinite set of strings of symbols drawn from a finite alphabet.
Machine Independence A term applied to software that is not dependent on the properties of a particular machine.
High-level Programming Language A programming language with strong abstraction from the details of the computer, such as Python or Java
Object-Oriented Programming (OOP) Object-oriented programming (OOP) is a computer programming model that organizes software design around data, or objects, rather than functions and logic.
Scripting Language A scripting language is a programming language that employs a high-level construct to interpret and execute one command at a time.
Data Types:
  • Integer

  • Float

  • String
  1. An integer is is a data type used to represent real numbers that do not have fractional values.

  2. A float is a variable type or variable definition that represents a real number.

  3. A string is a sequence of characters used to represent text. Once it's created it cannot be changed.
Primitive Data Type A primitive data type is a basic data type that is used to construct other data types.
Data Structure A specialized format for organizing, processing, retrieving and storing data.
Python Collection Data Structure:
  • List

  • Tuple

  • Set

  • Dictionary
  1. A list is an ordered arrangement of elements.

  2. a tuple is a data structure that is an ordered sequence of elements that are immutable, or unchangeable.

  3. A set is a data structure that stores unique elements of the same type in a sorted order.
  4. A dictionary is a collection which is ordered, changeable and do not allow duplicates.
Function A self contained module of code that accomplishes a specific task.
Local Variable A variable that is declared outside any function and is accessible to all routines in our program.
Global Variable A global variable is a variable that you can use from any part of a program, including within functions. It can be accessed throughout a program.
Source Program An original computer program written by a programmer that is converted into the equivalent object program, written in machine language, by the compiler or assembler.
Object Program A program into which some other program is translated by an assembler or compiler.
